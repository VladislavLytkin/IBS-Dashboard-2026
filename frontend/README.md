# IBS — Школьный дашборд (frontend)

Прототип веб-дашборда «Мониторинг школы» на **React + TypeScript + Vite**.
Интерфейс собран по макетам дизайна, работает на mock-данных (см. `src/data`).

> ⚠️ Аналитическая часть проекта (`Data Analysis/` — датасеты, ноутбуки, документация)
> к этому фронтенду **не относится и не изменяется**. Mock-данные специально вынесены
> в `src/data`, чтобы позже их можно было заменить на реальный API без переписывания UI.

## Запуск

```bash
cd frontend

# 1. установка зависимостей
npm install

# 2. запуск dev-сервера (http://localhost:5173)
npm run dev

# 3. production-сборка
npm run build

# 4. предпросмотр собранной версии
npm run preview

# (опционально) проверка типов
npm run typecheck
```

## Структура

```
src/
  components/   # Sidebar, Topbar, иконки, переиспользуемый UI
                #   (Card, Pagination, ParallelFilter, ComparisonTable, LevelBar, …)
  data/         # синтетические данные: classes, students, examResults, olympiadResults,
                #   attendance, activity, riskPredictions, …
  layouts/      # DashboardLayout (sidebar + topbar + контент)
  pages/        # по странице на каждый раздел дашборда
  styles/       # global.css — токены дизайна и компоненты
  types/        # доменные TypeScript-типы
  utils/        # scoring.ts (формула итогового рейтинга), riskModel.ts (ML-прогноз риска),
                #   random.ts (детерминированная генерация синтетики)
  router.tsx    # маршрутизация (react-router-dom)
```

## Логика расчётов

- **Итоговый рейтинг** — единая формула в `src/utils/scoring.ts`:
  `0.35·академ + 0.25·олимпиады + 0.15·посещаемость + 0.15·активность + 0.10·(100 − риск)`.
  Веса вынесены в константы, результат 0..100, сортировка по убыванию.
- **Прогноз риска** — прототип в `src/utils/riskModel.ts`:
  `0.35·снижение оценок + 0.30·пропуски + 0.20·низкая активность + 0.10·нет олимпиад + 0.05·отриц. динамика`.
  Уровни: 0–33 низкий, 34–66 средний, 67–100 высокий. **Это не реальная ML-модель.**
- Фильтр по параллелям (Все, 5–11) — в рейтинге классов, олимпиадах, рейтинге учеников и рисках.
